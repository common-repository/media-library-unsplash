== Description ==

Easily add Unsplash photographs to your website instantly without ever leaving WordPress!

The quickest and most straightforward method to add high-quality, free photographs to your WordPress media library is to use =Media Library Unsplash=, which you can do directly from your favourite stock photo community.

= Plugin Features =

* Search – In just a few seconds, you can locate and submit photographs for any topic with the Media Library Unsplash search!
* Filter – You can filter images by Latest, Popular, Oldest, View and Download!
* Time saver– From the comfort of your WordPress admin, quickly upload gorgeous stock photographs.
* Modify Metadata– Before adding an image to your media library, you may quickly change the image's filename, alt text, and caption.
* Caption – One click to import the image caption!
* Size – One click to import the image with custom size (Regular, Full, Small)!
* Download – You can download image in your local PC just one click!
* SEO – For screen readers, visually impaired users, and search engine optimisation, automatically include a pertinent alt description.
* Simple to Use – All you have to do is click an image to have it instantly added to your media library so you can use it on your website. It doesn't get any easier than that.
* API KEY – Easily set your Unsplash API Key.

== Supported Image Provider ==
* [Unsplash](http://unsplash.com/)

== Installation ==

= Installation =

1. Upload 'media-library-unsplash' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
